# SEALSGallery
GPS Tracker with Gallery and social functionality

Used technologies and patterns:
- MVVM
- Clean architecture
- Coroutines
- Coroutine Flow
- LiveData
- Firebase Realtime Database
- Google authentication
- Google Maps
- Room DB
- Room DB Type Converters
- Koin
- Gradle Secrets
- Picasso
- XML Design
- Kotlin Extension functions (mappers)
